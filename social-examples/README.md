# Social Examples

Reference images for Enablement's LinkedIn visual work.

See [`inspiration/`](inspiration/) - the canonical post-redesign reference set that drives [`../DESIGN_SOCIAL.md`](../DESIGN_SOCIAL.md). Read `inspiration/README.md` first - it annotates each reference image with what works, what could be improved, and what design rules each one teaches.

## Adding new examples

- **A new post-redesign reference** (e.g. a new cheat sheet or GIF that should drive the design system): add to `inspiration/` using the `NN_descriptor.ext` naming pattern and annotate it in `inspiration/README.md`.
- **Anything that doesn't fit** (e.g. competitor inspiration, brand mood boards): add a new subfolder with a clear name and a README explaining what's in it.
