01 / The register

Dense. Visualised. Premium.

Three words that must apply to every output. If a draft violates one of them, it's not done.

01
Dense
Every tile carries information. Empty space exists to let tiles breathe, never as filler.
02
Visualised
No tile is text-only. Each contains a mockup, mini-table, workflow snippet, or chart fragment that makes the concept legible without reading.
03
Premium
Subtle gradients, glassmorphic surfaces, soft drop shadows, faint grid textures. The graphic should look like a dashboard component, not a marketing poster.

03 / Typography

Sofia Sans. JetBrains Mono. That's it.

No serif typefaces anywhere. One italic-emphasis word per headline max. Mono eyebrows above section headings. Tabular numerics on all stats.

04 / Background

Gradient plus grid. Never flat.

Two stacked layers. Subtle gradient from --color-canvas top-left to a flow-blue tint bottom-right, plus a dot grid for depth. Should be barely perceptible at first glance, visible on close inspection.

05 / Bento tile

The main primitive. Always glassmorphic.

The unit of information in cheat sheets. White fill with a subtle vertical gradient overlay, 1px border (always present), 12px radius, soft shadow, minimum 24px internal padding. Optional 4px left-accent stripe for semantic state.

step 01
Default bento
White fill with subtle gradient overlay. Border always present. Minimum 24px padding. No left stripe.

4px left stripe in Enablement Red. Use for the hero tile in a cheat sheet, one per graphic max.

06 / Pills + chips

Mono eyebrows. Sans chips.

Pills carry category labels (mono uppercase). Chips carry tactical labels (sans regular). Both use the pill radius. Backgrounds use subtle gradients to sustain the premium feel.

07 / Stat callout

Numbers carry the post. Let them.

When a post hinges on one number, lead with the stat. Sofia Sans bold display size, mono caption label. For before/after pairs, use the arrow form. All numerics tabular for vertical alignment.

Every rule above traces back to one of these three references. Read them with the annotations in social-examples/inspiration/README.md for what works, what could be improved, and what each one teaches.

Cold Email Cheatsheet

The most polished cheat-sheet execution to date. Premium feel target. Subtle gradient shifts inside title boxes and pills. Strong show-don't-tell with mini mockups and tactic chips. Improvements: drop the serif subtitle, add more density via more bentos rather than more text per bento.

GTM Orchestration Framework
