# Enablement.ch — Design System

Live styleguide for Enablement.ch. Single self-contained HTML page; every token, component, and rule renders with the same CSS the site will ship.

**Live:** https://marcusaurelian.github.io/enablement-design-system/

## Stack
- Sofia Sans (Google Fonts) + JetBrains Mono
- Crimson `#E11E48` accent on a cool off-white canvas
- Light + dark mode via semantic CSS custom properties

## Edit
The whole system lives in `index.html`. Tokens are defined at the top of the `<style>` block. Sections are commented and numbered.
